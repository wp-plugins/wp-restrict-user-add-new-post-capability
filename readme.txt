This plugin will disable all non-admin users from creating new posts in your blog. The "add new" submenu will also be hidden in the user dashboard.


== Description ==

This plugin specifically disables all non-admin users (with posting capability), from creating and adding new POSTS from the user dashboard. However, all standard roles 
assigned to the respective user levels (mainly EDITOR and AUTHOR), will remain the same - un-affected. 


The "add new" submenu item will also be hidden from these users in the user's dashboard.


== Installation ==

1. Download the napr.zip file
2. Upload and extract the contents of napr.zip to your wp-content/plugins/folder
3. Activate the WP Restrict User 'ADD NEW POST' Capability in your WP-admin
4. Done! There is nothing else to this plugin.
